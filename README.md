# Project Layout

- `Task-1/`: WikiArt multi-task training, prediction, outlier review notebooks, and audit data.
- `Task-2/`: separate Task 2 work.
- `venv/`: shared virtual environment, linked to `Task-1/venv`.

Run Task 1 commands from `Task-1/`, or prefix paths from the repo root with `Task-1/`.
Use `venv/bin/python` from either the repo root or `Task-1/`.
